# Atelier IT - Système de Réservation

Système complet de gestion et de réservation de salles et d'équipements.

## Structure du projet

- **Backend** : NestJS + Prisma + SQLite
- **Frontend** : React + Vite + Tailwind CSS

## Installation

### Backend
1. `cd backend`
2. `npm install`
3. `npx prisma db push`
4. `npm run db:seed`
5. `npm run start:dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
